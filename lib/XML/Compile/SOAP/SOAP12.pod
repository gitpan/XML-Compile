=head1 DESCRIPTION

**WARNING** Implementation not finished: not usable!!

This module handles the SOAP protocol version 1.2.
See F<http://www.w3.org/TR/soap12/>).

=head1 DIAGNOSTICS

I<Error:> an input message does not have faults

I<Error:> headerfault does only exist in SOAP1.1

I<Error:> message is INPUT or OUTPUT, not $inout

I<Error:> option 'destination' only for writers

I<Error:> option 'mustUnderstand' only for writers

I<Error:> option 'roles' only for readers
